we predict how many confirmed cases is going to appeer in each county in the next 7 days

Using the ARIMA model we forecast new cases in the coming week.

The ARIMA model works by finding a linear regression based on the time series previous timelags (AR, auto-regression). To make the time series stationary we need to differentiate it (I, integrated). It also looks at the moving average (MA) when forecasting. We use a gridsearch function to find the appropriate orders for each county.
